== Installation ==
1. Upload `oauth-for-gap-messenger/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to https://my.gap.im and create OAuth Gap robot, and get client_id and Token
4. Setting up client_id and token in plugin setting, click save changes and get redirect_url
5. Set redirect link in https://my.gap.im Robot setting.
6. use short code `[login_gap]` in your theme.
7. Done!
